# klog.sh

[klog](https://github.com/kubernetes/klog) for bash (only bash for now).

Sourcing `klog.sh` will expose the following functions:

- `klog::Info ...`
- `klog::Infof format ...`
- `klog::Infoln ...`
- `klog::Warning ...`
- `klog::Warningf format ...`
- `klog::Warningln ...`
- `klog::Error ...`
- `klog::Errorf format ...`
- `klog::Errorln ...`
